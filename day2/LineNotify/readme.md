## 準備
- デスクトップに新しいフォルダを作成

## ライブラリを追加
- コマンドプロンプト/ターミナルで Desktop > 作成したフォルダ内に移動

```
$ cd Desktop
$ cd {作成したフォルダ名}
```

- 下記コマンドを順に実行

```
$ npm init -y
$ npm i --save axios
```

## プログラムをかく
- 同じフォルダ内に`app.js`を作成、プログラムをコピー
- `app.js`内の「const LINE_TOKEN = `トークンを指定`;」部分に、トークンを記入
- 保存

## 実行
```
$ node app.js
```
